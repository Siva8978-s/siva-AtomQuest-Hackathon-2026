# AtomQuest Frontend

React + Vite frontend for the AtomQuest Portal.

## Run

```powershell
npm run dev
```

Default URL:

```txt
http://localhost:5173
```

The API client expects the backend at:

```txt
http://localhost:5000/api
```

## Build

```powershell
npm run build
```

## Main Screens

- `/login` - role-based login
- `/employee` - employee goals and quarterly check-ins
- `/manager` - team goals, approvals, rework, and comments
- `/admin` - organization dashboard, charts, reports, users, and unlock actions
